# Receipt Maker

A web application for generating customizable receipts with live preview and export.
Built as a learning project using React and ASP.NET Core.

## Features

- User authentication
- Receipt templates
- Editable receipt content
- Logo upload and styling
- Live preview
- Download with watermark
- Watermark-free download after payment

## Tech Stack

**Frontend**
- React
- TypeScript

**Backend**
- ASP.NET Core Web API
- Entity Framework Core
